# Profile
Django Mini Project in which sqlite3 DB is used. I learned this project from you-tube channel Binary Beast. 

Please use virtual environment to run this project.

Install requirements from -

`pip install -r requirements.txt`

Run server :

`python manage.py runserver`

For profile go to :
`http://127.0.0.1:8000/`

For blogs go to :
`http://127.0.0.1:8000/blog`
